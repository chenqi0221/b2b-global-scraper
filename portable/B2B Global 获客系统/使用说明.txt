使用说明
========
1. 双击 "B2B Global 获客系统.exe" 启动程序
2. 程序会自动在后台启动 Python 后端服务
3. 首次启动可能需要 10-30 秒初始化

注意事项
========
- 请勿移动或删除 backend.exe 文件
- 两个 exe 文件必须放在同一个目录下
- 如遇到防火墙提示，请允许访问网络
- 程序仅监听 127.0.0.1:8756（本机），不会暴露到外网

文件说明
========
- B2B Global 获客系统.exe : 主程序（前端界面）
- backend.exe : 后端 API 服务（自动启动，无需手动操作）
