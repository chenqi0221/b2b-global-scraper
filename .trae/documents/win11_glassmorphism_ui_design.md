# B2B Global 获客系统 - Win11 毛玻璃UI设计文档

## 设计目标
- 采用 Windows 11 风格的毛玻璃（Glassmorphism）设计
- 侧边栏可折叠，只显示图标
- UI 自适应窗口大小
- 标题改为 "B2B Global 获客系统"

## 视觉风格

### 色彩方案
- **主背景**: `#0F172A` (深蓝黑，带透明)
- **毛玻璃面板**: `rgba(30, 41, 59, 0.7)` + `backdrop-filter: blur(20px)`
- **侧边栏**: `rgba(15, 23, 42, 0.85)` + 模糊
- **卡片背景**: `rgba(255, 255, 255, 0.05)` + 边框 `rgba(255,255,255,0.1)`
- **主色调**: `#3B82F6` (亮蓝)
- **强调色**: `#10B981` (翠绿)
- **文字主色**: `#F1F5F9` (灰白)
- **文字次色**: `#94A3B8` (灰蓝)

### 字体
- **标题**: "Segoe UI", 18px, bold
- **正文**: "Segoe UI", 11px
- **等宽**: "Consolas", 10px (日志)

### 圆角与阴影
- **卡片圆角**: 12px
- **按钮圆角**: 8px
- **面板阴影**: `0 8px 32px rgba(0,0,0,0.3)`
- **边框**: 1px solid `rgba(255,255,255,0.08)`

## 布局架构

### 整体结构
```
+--------------------------------------------------+
|  [折叠按钮]  B2B Global 获客系统        [最小化] |
+----------+---------------------------------------+
|          |                                       |
| 侧边栏    |          主内容区                      |
| (可折叠)  |         (自适应)                       |
|          |                                       |
+----------+---------------------------------------+
```

### 侧边栏设计
- **展开状态**: 宽度 200px，显示图标 + 文字
- **折叠状态**: 宽度 60px，只显示图标
- **折叠按钮**: 顶部有一个 `<` / `>` 切换按钮
- **背景**: 半透明深色 + 右侧 1px 分隔线

### 主内容区
- **状态卡片**: 顶部横向排列，圆角卡片
- **工作区**: 中间可滚动区域
- **底部状态栏**: 显示当前地区、状态信息

## 组件规范

### 毛玻璃卡片
```python
# 效果
bg = rgba(255, 255, 255, 0.05)
border = 1px solid rgba(255,255,255,0.1)
border_radius = 12px
backdrop_filter = blur(10px)
```

### 按钮样式
- **主按钮**: 蓝色渐变 + 悬停发光
- **次按钮**: 透明背景 + 白色边框
- **危险按钮**: 红色渐变

### 输入框
- 透明背景
- 底部边框高亮
- 聚焦时蓝色发光

## 响应式规则
- **窗口 < 900px**: 侧边栏自动折叠
- **窗口 < 600px**: 简化布局，卡片堆叠
- **窗口 > 1400px**: 侧边栏可固定展开

## 动画效果
- 侧边栏折叠: 300ms ease-in-out
- 页面切换: 200ms fade
- 按钮悬停: 150ms scale(1.02)
- 卡片悬停: 边框高亮
